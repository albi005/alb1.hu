Albi

DisHelps

The DisHelps app, that includes a .NET MAUI cross-platform mobile app and an ASP.NET Core Web API. The server is hosted on Azure App Service and persists data in an Azure SQL database using Entity Framework Core. Static content, like the knowledge base can be edited using the Piranha CMS interface on the web. Authentication is handled using Azure Active Directory B2C.

MaterialColorUtilities

C# implementation of Google's Material color utilities with thousands of downloads on NuGet. Includes all of the algorithms required for adding Material You colors to a .NET app. Can be used to generate a color palette with perfect contrasts from a single color extracted from an image.

Menza

A website where students can quickly check what is for lunch and vote on it. Made with Blazor WebAssembly and some JavaScript. The source code is available here.

Blace

Reddit's r/place recreated with Blazor WebAssembly as the client, ASP.NET for the server and SignalR for real-time communication between them. For the source code, click here.

Keleti Kúria

keletikuria.azurewebsites.net

A simple Blazor Server website for a mansion in Hungary. Users can learn about the mansion's history, see photos, check its availability and book it. Content is stored and edited using Piranha CMS. Authentication is handled by ASP.NET Core Identity.
